# Снимки GHS, книга калибровок KIMTRANS и расчёты РД.3036 / РД.3035

В корне репозитория лежат файлы `*.webp` с именами UUID — это **фотографии страниц** из двух источников:

1. **Книга калибровок цистерн (Tank Calibration Table)** для серии **KIMTRANS SPB 3209–3213, 3218**, таблицы из **GHS 9.10B** (дата на оттисках 06-05-10): глубина (sounding), объём, масса, **LCG/TCG/VCG**, **GML**, **FSM**.
2. **Расчёты остойчивости при обледенении** по правилам РМРС: **РД.3036-901-СФ.001** (случаи нагрузок с обледенением, диаграммы GZ, сводки критериев) и **РД.3035-901-СФ.002** (нагрузка обледенения, парусность при *d* = 1,72 м, схемы палуб).

Ниже — структурированное содержание **без необходимости открывать каждый файл**. Полные построчные таблицы по-прежнему только на оригиналах снимков или в PDF буклета.

## 2. Общие условия таблиц GHS на снимках

- Программа: **GHS 9.10B**; на оттисках: **06-05-10 09:45:17** (где указано).
- Режим: **No Trim, No Heel**.
- Для топлива и балласта в шапках указана **проницаемость 0,98** (пресная — также 0,98), для носового пика балласта — **0,95**.
- Столбцы: **Snding (мм)** → **Volume (m³)** → **Weight (т)** → **LCG / TCG / VCG (м)** → **GML (м)** → **FSM (м·т)**.
- Нулевой замер: **Zero Sounding is at the Reference Point** — координаты опорной точки даны внизу каждой страницы.

## 5. РД.3036-901-СФ.001 — смысл снимков

- **Связь с буклетом:** расчёт остойчивости при обледенении опирается на формуляр / буклет (в т.ч. **3035-0006**), данные обледенения — из **РД.3035-901-СФ.002**.
- **Судно в тексте вводной (лист 2):** самоходный плашкоут, *L* ≈ 97,536 м, *B* ≈ 24,384 м, *D* ≈ 6,096 м, осадка в примере **4,439** м (как на снимке; приложение «РЕЙД-8» в коде может отличаться деталями — сверять с PDF буклета).
- **Случай 02 с обледенением (табл. 1-1, лист 3):** постоянный вес, груз, балласт **0 %**, топливо и пресная **10 %**, обледенение **78,28** т, **Δ ≈ 8832,80** т, **VCG ≈ 6,54** м, **X_g ≈ 46,76** м от НП. Танки в списке: **BW-02**, **BW-01**, **FORE PEAK SW.C**, **FUEL-PTS.P / STB**, **FRESH-PTS.P / STB**.
- **Случай 02 — последующие листы:** гидростатика и **GZ(θ)** при **Δ = 8832,8** т (лист 4), **GM₀ ≈ 7,69** м (как исправленная метацентрическая на листах с таблицами 1-4), сводка **табл. 1-6** (лист 6): погодный критерий **K = b/a**, фактическое **2,03** при требовании **≥ 1**; статический крен **~0,17°**; **GZ** при максимуме **~1,57** м; площади под кривой GZ до 30°/40° и между 30°–40°; **GM** в таблице критериев **7,69** м.
- **Случай 04 с обледенением:** **Δ ≈ 3307,56** т (табл. 2-1), балласт **100 %** (в т.ч. носовой пик **101,62** т при **LCG ≈ 93,102** м), **VCG ≈ 5,38** м. Листы 8–10: **KMT ≈ 30** м, теоретическая осадка **~1,75** м, таблица **GZ** до 60°, **табл. 2-6**: **K ≈ 2,87**, **GZ_max ~5,65** м, **GM ~24,6** м в сводке — это **другой режим загрузки**, не смешивать с полным грузом без проверки.

## 6. РД.3035-901-СФ.002 — обледенение

- Проект **3035**, суда **«Рейд-7» … «Рейд-12»**, правила РМРС (редакция на снимке — 2023).
- Парусность при **d = 1,72** м: суммарная парусность **760,43** м², центры **X ≈ 46** м**, Z ≈ 5,94** м (от базовой линии) — см. таблицу на листе 4.
- Таблицы **3.2–3.3**: лёд на горизонтальных проекциях палуб (**66,87** т) + боковая поверхность (**11,41** т) → **итого 78,29** т, **X ≈ 45,67** м, **Z ≈ 6,36** м.
- Схема с номерами зон **1–18** на рубке, баке и главной палубе — для привязки строк таблицы 3.2.
