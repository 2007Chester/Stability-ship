# Приложение и модули

## Запуск

Из каталога `stability_app/`:

```bash
python3 -m streamlit run app.py
```

Зависимости: `stability_app/requirements.txt`.

## Вкладки

1. **Остойчивость** — ввод масс по **цистернам буклета** (LCG/KG жидкости только из разд. 6 в `tank_booklet.py`), снабжению, **расходным** ёмкостям, углю; порожнее из **`ship_data`**; сайдбар: θзал, ПВСВ, GG₀, LCF, режим **X от миделя**; итоги **Δ**, **GM**, критерии **ИМО A.749**, график **GZ**, таблица в стиле Excel.
2. **Груз в трюмах по осадкам** — оценка массы насыпного груза по осадкам нос/корма; порожнее судна и судовое снабжение совпадают с первой вкладкой и `ship_data` (отдельного поля «прочее» нет).

## Основные модули

| Модуль | Назначение |
|--------|------------|
| `app.py` | UI Streamlit |
| `ship_data.py` | Константы судна, загрузка `_embedded.json` |
| `stability.py` | Гидростатика, GZ, GM, ИМО A.749, осадки/груз по черновику |
| `tank_booklet.py` | Список цистерн разд. 6 |
| `excel_ui.py` | Таблица Trim, стили, преобразования X |
| `cargo_excel_data.py` | Эталон Excel и пресет `PRESET_V2_GRUZ` |
| `calc_state.py` | Сохранение/загрузка расчёта JSON |
| `rs_criteria.py` | Доп. критерии/формулы Excel-стиля |

## Сохранение расчёта

В сайдбаре: скачать **JSON** со всеми полями или загрузить ранее сохранённый файл.

## Тесты

Каталог `stability_app/tests/` — юнит-тесты для таблиц, остойчивости, состояния расчёта.
