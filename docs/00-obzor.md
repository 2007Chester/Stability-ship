# Обзор

## Назначение проекта

Репозиторий содержит:

1. **Данные и формулы** для оценки **остойчивости** баржи типа **Kimtrans SPB 3210** в исполнении **«РЕЙД-8»** по материалам буклета остойчивости (номер в коде: **3035-0006**, дифферент 0, плотность морской воды **1,025 т/м³**).
2. **Веб-приложение** (Streamlit) для ввода загрузки по цистернам буклета, снабжению, расходным ёмкостям, углю; вывод **GM**, **критериев ИМО A.749**, кривой **GZ**, оценки осадок и дифферента.
3. **Вторую вкладку** — оценка массы насыпного груза в трюмах по осадкам и введённым массам жидкостей/балласта.

## Источник истины для чисел в коде

| Данные | Где в проекте |
|--------|----------------|
| Размерности, порожнее, ρ | `stability_app/ship_data.py` → `SHIP` |
| Гидростатика T–Δ–KB–KMT и матрица GZ₀ | `stability_app/_embedded.json` (выгрузка из буклета) |
| Цистерны разд. 6 (LCG от кормы, KG, ориентир 100%) | `stability_app/tank_booklet.py` → `BOOKLET_TANKS` |
| Эталон строки «В грузу» из Excel | `stability_app/cargo_excel_data.py` |

## Чего эта документация не заменяет

- **Официальный буклет** (`Буклет остойчивости_rus.pdf`) — юридически значимый документ, чертежи, полные таблицы.
- **Конвенция о грузовой марке** — нормативный контекст загрузки.
- **Скриншоты GHS** (файлы `*.webp` в корне) — детальные **калибровочные таблицы** отдельных цистерн в программе General Hydrostatics; их не переписывали в текст: см. [07-fajly-v-korne-repo.md](07-fajly-v-korne-repo.md).

## Связь координат X

В приложении по умолчанию **X центра тяжести груза/жидкости** задаётся **от миделя, «+ к носу»** (как в типичном Trim/Stability General). Для расчёта моментов и LCG используется приведение к **расстоянию от шп. кормы вперёд** (LCG): см. разделы в [02-gidrostatika-i-krivye-gz.md](02-gidrostatika-i-krivye-gz.md) и модуль `excel_ui.py`.
