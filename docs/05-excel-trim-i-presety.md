# Excel «Stability General / Trim» и пресеты приложения

## Снимок таблицы «В грузу»

Файл **`stability_app/cargo_excel_data.py`** содержит:

- **`ROWS_CARGO_IN_GRUZ`** — строки как на листе **«В грузу»** рабочей книги: наименование, масса, **Xг** (обычно **от миделя, + к носу**), **KG** над килем.
- Используется в **тестах** (`tests/test_cargo_in_gruz.py`) для проверки сумм, LCG и гидростатики при заданных условиях.

Строка «Судно порожнее» в этом снимке (**2210 т** и т.д.) — **исторический** вариант; в приложении порожнее берётся из **`ship_data.SHIP`**, а не из этой строки.

## Пресет «В грузу (из Excel)»

Словарь **`PRESET_V2_GRUZ`** — начальные значения полей вкладки остойчивости, согласованные с Excel:

| Смысл Excel | Поля приложения |
|-------------|-------------------|
| Судовое снабжение | `stab_m_stores`, `stab_x_stores`, `stab_kg_stores` |
| Топливо 6 т, X −46, KG 1,2 | **Расходные** цистерны: `stab_m_fuel_svc=6`, `stab_x_fuel_svc=-46`, `stab_kg_fuel_svc=1,2`; **не** дублируется в FUEL-PTS/STB (`stab_t0`/`stab_t1` = 0) |
| Уголь | `stab_m_coal`, `stab_x_coal`, `stab_kg_coal` |
| Балласт Л/Б 67 т / ПР/Б 69 т | Массы в танках **7** и **8** (BW-02 порт / прав): `stab_t7=67`, `stab_t8=69` |

**LCG/KG жидкости** в расчёте для танков **0–8** берутся из **буклета** (`BOOKLET_TANKS`), а не из упрощённых X = −37,8 из Excel для балласта.

## Рабочий файл Excel в корне

**`Рабочая Stability General (Без груза).xls`** — отдельная рабочая книга; не импортируется в код автоматически. При изменении методики обновляют **`cargo_excel_data.py`** и эту документацию.
