# Гидростатика и кривые GZ

## Встроенные таблицы (`_embedded.json`)

Файл `stability_app/_embedded.json` содержит:

### 1. Массив `hydro`

Каждая строка: **[T, Δ, KB, KMT]** где:

- **T** — осадка от ОП, м (дифферент 0 в таблице буклета);
- **Δ** — водоизмещение, т;
- **KB** — аппликата центра плавучести над килем, м;
- **KMT** — поперечная метацентрическая высота над килем (для поперечной метацентрической оценки), м.

В коде: `HYDRO_T`, `HYDRO_DISP`, `HYDRO_KB`, `HYDRO_KMT` — используются для интерполяции **средней осадки по Δ** и **KMT по Δ**.

**Диапазон осадок в таблице:** от **1,20 м** до **4,40 м** (шаг 0,05 м); соответствующие Δ примерно от **~2216 т** до **~8975 т** (см. JSON).

### 2. Углы и матрица GZ₀

- **`angles`**: углы крена в градусах — **5, 10, 15, …, 60** (шаг 5° до 60°).
- **`gz_rows`**: каждая строка — один **водоизмещение** (первое число в строке = Δ, т) и далее значения **GZ₀** в метрах для каждого угла из `angles` при **VCG = 0** (от КН), как в буклете.

Интерполяция: по **Δ** между строками и по **φ** между углами — функция `gz0_at()` в `stability.py`.

### 3. Формула плеча остойчивости

Для заданного фактического **VCG над килем** (в приложении — итоговый **KG**, с учётом GG₀ и т.д. по смыслу расчёта):

**GZ(φ) = GZ₀(Δ, φ) − (KG + FSC) · sin(φ)**

где **KG** — аппликата центра масс над килем (в приложении — итоговый **KG₀**, с GG₀ из сайдбара при необходимости), **FSC** — вводимая потеря GM от ПВСВ (м). Добавление **FSC** к множителю при sin φ согласует начальный наклон кривой с **GM = KMT − KG − FSC** и с красной касательной **GM·φ** (φ в радианах) на графике. При **FSC = 0** формула совпадает с **GZ₀ − KG·sin φ**.

### 4. Начальная метацентрическая высота

**GM = KMT(Δ) − KG − FSC**

- **KMT** — из гидростатики по Δ;
- **KG** — итоговый центр масс загрузки;
- **FSC** — потеря GM из‑за ПВСВ (вводится в сайдбаре, м).

## Продольный дифферент (оценка в приложении)

Для осадок нос/корма и дифферента в **буклете нет** таблиц MCTC/LCF в том виде, как в судне с полными линиями. В `stability.py` используется **упрощённая модель** прямоугольной ватерлинии: **MCTC**, момент от **Δ·(LCG − LCF)**, осадки нос/корма от средней осадки. Это **оценка**; при большом дифференте оговорка из буклета о таблицах GZ при нулевом дифференте остаётся актуальной.

## Координата X в таблице загрузки

- **От миделя (+ к носу):** LCG от кормы = **LBP/2 + X_мидель**.
- **От кормы:** LCG = введённое X.

Переводы реализованы в `excel_ui.py`: `x_g_to_from_ap`, `x_table_from_lcg_ap`, `table_x_from_lcg_ap`.
